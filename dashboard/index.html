📋 Command Cheatsheet

🤖 Scraper
  • @scraper scan self or @scraper scan . Analyze the local project code
  • @scraper scan <path> Analyze a specific local folder
  • @scraper status Show last scan summary
🤖 Keeper
  • @keeper recall <subject> Search facts on a subject (e.g. project-ALLY)
  • @keeper list List the latest stored facts
🤖 Reconciler
  • @reconciler detect Scan Keeper's DB & run AI conflict analysis
  • @reconciler status Show open & resolved conflicts
  • @reconciler resolve <conflict_id> <fact_id> [reason] Manually pick a winning fact
🤖 Registry
  • @registry register name=X skills=X,Y,Z description=... Register an agent in the directory
  • @registry discover <skill> Find agents by capability
  • @registry list List all registered agents
🌉 Bridge
  • Passive observer — captures all room messages for the dashboard. No commands.
