# Cover Image Prompt

## Style
- **Aspect ratio**: 16:9 (1920×1080)
- **Style**: Dark tech/cyberpunk with warm accent colors — deep navy background, neon orange and cyan highlights
- **Format**: Digital illustration / concept art

## Prompt

```
Three ethereal robot agents floating in a dark digital void, connected by glowing threads of light forming a triangular mesh network. Each agent has a distinct silhouette: one holds a directory scroll (Registry), one guards a crystalline database (Keeper), one wields a balancing scale (Reconciler). In the center, a pulsating golden node labeled "Truth" emits concentric rings of data. Below them, a split screen shows code on one side and documentation on the other — contradictions highlighted in red. Style: cyberpunk aesthetic, deep navy background (hex #0a0e27), neon orange (#ff6b35) and cyan (#00d4ff) accents, volumetric lighting, 8K detail, Unreal Engine 5 render, cinematic composition, wide angle lens. No text overlays.
```

## Generation Notes
- Use Midjourney v6, DALL-E 3, or Stable Diffusion 3 with this prompt
- If the tool supports negative prompting, add: "realistic, photograph, cluttered, messy, text, watermark"
- The three agents should be clearly distinct to visually communicate the 3-agent architecture
- The code-vs-documentation split at the bottom communicates the problem space
- The golden "Truth" node in the center represents the reconciliation goal
